# Phone-shop
Aplicación frontend desarrollada con Angular que simula un catálogo de productos con navegación, búsqueda, detalle de producto y gestión de carrito.

## Características

- Listado de productos
- Vista detalle de producto
- Navegación mediante breadcrumbs
- Búsqueda de productos
- Selección de variantes (color, almacenamiento...)
- Añadir productos al carrito
- Persistencia del carrito en el almacenamiento del cliente

## 🛠️ Tecnologías utilizadas

- Angular
- TypeScript
- HTML5
- CSS / SCSS
- Jasmine (tests pendientes)
